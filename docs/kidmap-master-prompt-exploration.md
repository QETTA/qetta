# KidMap 앱 개선 — 플랜 모드 전 마스터 프롬프트 탐색

> **목적**: Plan Mode 진입 전, AI가 KidMap 앱의 현재 상태·목표·제약 조건을 완전히 이해하도록 하는 탐색 프롬프트
> **작성일**: 2026-02-01
> **버전**: 1.0
> **상위 플랫폼**: QETTA (qetta-super-model v3.0 기반)

---

## 0. 이 문서의 사용법

```
1. 아래 마스터 프롬프트를 Claude에게 전달
2. Claude가 질문 목록으로 응답 → 답변 제공
3. 충분한 컨텍스트 확보 후 "Plan Mode 진입" 요청
4. Claude가 구조화된 개선 계획 출력
```

## 3. 프롬프트 체이닝 가이드

```
[Step 1] 마스터 탐색 프롬프트 전달
    ↓
[Step 2] Claude 질문에 답변 (2-3회 왕복)
    ↓
[Step 3] 탐색 결과 요약 확인
    ↓
[Step 4] "Plan Mode 진입 준비도" 체크리스트 확인
    ↓
[Step 5] 6/6 달성 시 → Plan Mode 전환 프롬프트:

    "탐색 결과를 기반으로 Plan Mode로 전환한다.
     다음 형식으로 실행 계획을 수립하라:
     1. Phase별 마일스톤 (4주 단위)
     2. 각 Phase의 구체적 태스크 (Story 단위)
     3. 기술 설계 문서 (Architecture Decision Records)
     4. 테스트 전략
     5. 배포 전략"
```

## 4. 도메인 확장: KidMap 전용 BLOCK 구상

QETTA의 Block Engine을 KidMap에 적용할 때의 BLOCK 구조:

| BLOCK ID | 이름 | 설명 | 템플릿 |
|----------|------|------|--------|
| PHYSICAL | 신체발달 | 키, 몸무게, 대근육, 소근육 | 성장 곡선 보고서, 운동능력 평가서 |
| COGNITIVE | 인지발달 | 언어, 수리, 문제해결, 기억력 | 인지 발달 평가서, 학습 준비도 보고서 |
| SOCIAL | 사회성발달 | 또래 관계, 감정 조절, 협동 | 사회성 발달 보고서, 행동 관찰 기록 |
| LANGUAGE | 언어발달 | 표현언어, 수용언어, 문해력 | 언어 발달 평가서, 어휘력 추적 보고서 |
| CREATIVE | 창의성발달 | 미술, 음악, 상상놀이, 탐구 | 창의성 평가 보고서 |
| EMOTIONAL | 정서발달 | 자기인식, 감정표현, 자존감 | 정서 발달 보고서, 기질 분석서 |

### BLOCK 조합 예시

```
[PHYSICAL] + [COGNITIVE] + [LANGUAGE]
→ "만 4세 종합 발달 보고서" 자동 생성 (45초)

[SOCIAL] + [EMOTIONAL]
→ "사회정서 발달 평가서" 자동 생성 (30초)

전체 6 BLOCK 조합
→ "종합 발달 리포트" (전문가용/부모용 분리 출력)
```

## 5. 참고: QETTA 기존 프롬프트 패턴

KidMap 프롬프트 작성 시 참고할 QETTA의 프롬프트 설계 패턴:

| 패턴 | QETTA 적용 | KidMap 적용 |
|------|-----------|-------------|
| System Prompt + Domain Extension | `QETTA_SYSTEM_PROMPT` + `DOMAIN_PROMPTS[engine]` | `KIDMAP_SYSTEM_PROMPT` + `DEVELOPMENT_PROMPTS[block]` |
| Block Engine → Prompt Generator | `prompt-generator.ts` | `kidmap-prompt-generator.ts` |
| Super Model (SSoT) | `qetta-super-model.json` | `kidmap-super-model.json` |
| Inline Commands | `/분석`, `/요약`, `/번역` | `/발달분석`, `/보고서`, `/비교` |
| Context-Aware AI | `intelligent-context.ts` | `development-context.ts` |
