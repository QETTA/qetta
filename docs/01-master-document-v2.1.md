> **버전**: v2.1 (2026-01-30)
> **대상**: 경영진, 투자자, 전략 파트너

## Executive Summary

QETTA는 **정부지원사업 제안서 자동화 플랫폼**입니다.

| 핵심 가치 | 지표 |
|----------|------|
| **시간 단축** | 93.8% (3일 → 30분) |
| **비용 절감** | 90% (300만원 → 30만원) |
| **성공률 향상** | 학습 기반 피드백 |

### v2.1 핵심 변화

```
v2.0 (12개 Industry Block)
    ↓
v2.1 (10개 Industry Block, KSIC 기반 재정의)
    + 3-Layer Block Engine
    + Mem0 패턴 압축 (80% 토큰 절감)
    + 가격 티어 현실화
```

---

## 1. 문제 정의

### 1.1 시장 현황

한국의 정부지원사업 생태계:

| 항목 | 수치 |
|------|------|
| 연간 정부지원사업 | 3,000+ 건 |
| 연간 총 지원 예산 | 5조원+ |
| 중소기업 신청 건수 | 63만+ 건/년 |
| 평균 선정률 | 15-25% |

### 1.2 핵심 고충 (Pain Points)

**공급기업 (AI/SW 기업)**:
- 동일 내용 반복 작성 (매년 5-10건)
- 담당자 퇴사 시 노하우 유실
- 탈락 사유 분석 어려움

**수요기업 (제조업)**:
- 복잡한 공고문 해석
- 업종별 전문 용어 이해 부족
- 자격 요건 자가 진단 어려움

### 1.3 경쟁 환경

| 솔루션 | 한계점 |
|--------|--------|
| **컨설팅 업체** | 건당 300만원+, 인력 의존 |
| **문서 자동화** | 맞춤화 부족, 규정 미반영 |
| **범용 AI** | 도메인 지식 부족 |

## 2. 솔루션

### 2.1 QETTA 3-Layer Block Engine

```
┌─────────────────────────────────────────────────────────────┐
│                    QETTA 3-Layer Architecture                │
├─────────────────────────────────────────────────────────────┤
│  Layer 3: Session Context (500 tokens)                      │
│  └─ 현재 대화, 의도 분석, 문서 생성 요청                       │
├─────────────────────────────────────────────────────────────┤
│  Layer 2: Company Block (1,500 tokens, Mem0 압축)           │
│  └─ 기업 프로필, 신청 이력, 탈락 패턴, 성공 패턴               │
├─────────────────────────────────────────────────────────────┤
│  Layer 1: Domain Engine (2,000 tokens)                      │
│  └─ Industry Block (10개) + Program Block (정부사업 규정)    │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 10개 Industry Block (v2.1)

| Block | 한글명 | AI바우처 비중 | Color |
|-------|--------|--------------|-------|
| FOOD | 식품/음료 | 15% | orange |
| TEXTILE | 섬유/의류 | 8% | pink |
| METAL | 금속/철강 | 12% | slate |
| CHEMICAL | 화학/소재 | 10% | amber |
| ELECTRONICS | 전자/반도체 | 18% | cyan |
| MACHINERY | 기계/장비 | 14% | blue |
| AUTOMOTIVE | 자동차/부품 | 8% | indigo |
| BIO_PHARMA | 바이오/제약 | 7% | rose |
| ENVIRONMENT | 환경/에너지 | 5% | emerald |
| GENERAL | 일반제조 | 3% | gray |

### 2.3 핵심 기능

#### 자동 제안서 생성
- Industry Block 기반 전문 용어 자동 삽입
- Program Block 기반 규정 준수 검증
- Company Block 기반 맞춤형 컨텐츠

#### 탈락 패턴 학습
- 과거 탈락 사유 분석 및 패턴화
- 유사 사업 지원 시 사전 경고
- 성공 패턴 기반 권장사항 제시

#### 자격 요건 자동 검증
- 공고문 파싱 및 요건 추출
- 기업 프로필 대비 자동 매칭
- 적합도 점수 산출 (0-100)

## 4. 시장 분석

### 4.1 TAM/SAM/SOM

```
TAM (Total Addressable Market)
└─ 전체 중소기업 문서 자동화: 5,000억원

SAM (Serviceable Addressable Market)
└─ 정부지원사업 문서 자동화: 1,000억원

SOM (Serviceable Obtainable Market)
└─ AI/제조 분야 초기 시장: 100억원 (Year 3)
```

### 4.2 타겟 고객

**1차 타겟: AI 공급기업** (2,000+ 등록업체)
- NIPA AI바우처 공급기업
- 데이터바우처 공급기업
- 클라우드바우처 공급기업

**2차 타겟: 제조 중소기업** (60,000+ 스마트공장 대상)
- 스마트공장 도입 기업
- AI 적용 제조업
- 수출 제조기업

**3차 타겟: 정책 컨설팅** (500+ 업체)
- 정부사업 전문 컨설팅
- 회계법인/특허법인
- 테크노파크, 창조경제혁신센터
