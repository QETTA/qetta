FAQ

자주 묻는 질문

아닙니다. QETTA는 e나라도움에 제출하기 직전 증빙과 정산서의 품질을 검수하는 레이어입니다. 기존 시스템을 그대로 쓰면서, 반려 가능성을 사전에 차단합니다.
증빙 내 PII(주민번호, 계좌번호, 연락처)는 업로드 즉시 자동 탐지 후 마스킹 처리됩니다. 원본은 국내 서버에 암호화 분리 저장하며, 처리 로그를 실시간 제공합니다. 개인정보보호법(PIPA) 기준을 준수합니다.
파일럿 3건은 완전 무료이며, 카드 등록도 필요 없습니다. 파일럿 종료 후 별도 계약 없이는 과금이 발생하지 않습니다. 효과를 확인한 뒤 유료 전환 여부를 결정하시면 됩니다.
QETTA는 자문이 아닌 검수 보조 도구입니다. 자동 제출 기능이 없으며, 모든 산출물은 담당 전문가가 직접 검토·승인 후 제출합니다. 리퍼럴 모델 또한 감사/세무 독립성에 영향을 주지 않는 구조입니다.
파일럿 단계에서는 주요 부처(중기부, 복지부, 과기부)의 정산 서식을 우선 지원합니다. 법인 커스텀 룰(Pro 이상)을 통해 특정 부처 서식을 추가할 수 있으며, 정식 출시 시 대부분의 e나라도움 등록 부처를 커버할 예정입니다.
