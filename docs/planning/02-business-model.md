# QETTA 비즈니스 모델 v2.0

## 가격 체계 설계

### 개요

QETTA의 가격 체계는 **4티어 구독 + EXPORT 크레딧 하이브리드** 모델입니다. 이 설계의 핵심 원칙은 다음과 같습니다.

첫째, 낮은 진입 장벽입니다. FREE 티어로 제품을 먼저 경험하게 합니다. 둘째, 자연스러운 업그레이드 경로입니다. 사용량이 늘면 자연스럽게 다음 티어로 이동합니다. 셋째, 고수익 레이어 분리입니다. EXPORT는 크레딧으로 분리하여 ARPU를 높입니다.

### 가격 체계 구조도

## 티어별 상세 설계

### FREE 티어

FREE 티어의 목적은 **고객 획득(Lead Generation)**입니다. 무료로 제품의 핵심 가치를 경험하게 하되, 본격적인 사용에는 제한을 둡니다.

**가격 및 기간**

```typescript
interface FreeTier {
  price: 0;
  duration: "unlimited";  // 기간 제한 없음 (30일 제한 제거)
  
  // 왜 무제한인가?
  // - 30일 제한은 "급하게 결정해야 한다"는 압박 → 이탈 유발
  // - 무제한은 "천천히 써보세요" → 습관 형성 → 자연스러운 전환
}
```

**사용량 제한**

| 항목 | 제한 | 업그레이드 트리거 |
|-----|-----|-----------------|
| 월간 문서 생성 | 5건 | "이번 달 한도에 도달했습니다" |
| BLOCK | 1개만 선택 | "추가 BLOCK이 필요하신가요?" |
| 저장 공간 | 100MB | "저장 공간이 부족합니다" |
| 팀원 | 1명 (본인만) | "팀원을 초대하시겠습니까?" |
| AI 생성 횟수 | 월 20회 | "AI 생성 횟수를 초과했습니다" |

**기능 제한**

```typescript
interface FreeTierFeatures {
  // 활성화된 기능
  documentGeneration: true;      // 기본 문서 생성
  hashVerification: true;        // 해시 검증
  basicTemplates: true;          // 기본 템플릿
  
  // 비활성화된 기능
  exportCredits: false;          // EXPORT 기능
  prioritySupport: false;        // 우선 지원
  apiAccess: false;              // API 연동
  teamFeatures: false;           // 팀 기능
  advancedAnalytics: false;      // 고급 분석
}
```

**전환 유도 전략**

FREE 사용자가 자연스럽게 PRO로 전환되도록 다음과 같은 터치포인트를 설계합니다.

1. **한도 도달 시**: "이번 달 5건 한도에 도달했습니다. PRO로 업그레이드하면 월 50건까지 생성할 수 있습니다."
2. **추가 BLOCK 필요 시**: "SEMICONDUCTOR BLOCK을 사용하시려면 PRO 이상이 필요합니다."
3. **EXPORT 시도 시**: "글로벌 입찰 문서는 EXPORT 크레딧이 필요합니다. PRO부터 사용 가능합니다."

### PRO 티어 (핵심 수익 티어)

PRO 티어는 **PMF 검증의 핵심**입니다. 월 9.9만원은 "결재 없이 법인카드로 긁을 수 있는 금액"이며, 개인 사업자나 소기업이 부담 없이 시작할 수 있는 가격대입니다.

**가격**

```typescript
interface ProTierPricing {
  monthly: 99000;           // 월 9.9만원
  yearly: 990000;           // 연 99만원 (2개월 무료, 17% 할인)
  
  // 연간 결제 유도
  yearlyDiscount: "2개월 무료";
  yearlyBadge: "BEST VALUE";

```

**사용량**

| 항목 | 제한 | FREE 대비 |
|-----|-----|----------|
| 월간 문서 생성 | 50건 | 10배 |
| BLOCK | 3개 선택 | +2개 |
| 저장 공간 | 5GB | 50배 |
| 팀원 | 3명 | +2명 |
| AI 생성 횟수 | 월 500회 | 25배 |

**기능**

- 소규모 스타트업 (직원 10명 미만)

### TEAM 티어

TEAM 티어는 **확장과 Lock-in**을 위한 티어입니다. 한 회사에서 여러 명이 사용하면 다른 툴로 전환하기 어려워집니다.

**가격**

```typescript

  monthly: 290000;          // 월 29만원
  yearly: 2900000;          // 연 290만원 (2개월 무료)
  
  // 사용자당 추가 비용
  additionalUserPrice: 30000;  // 10명 초과 시 인당 3만원/월

```

**사용량**

| 항목 | 제한 | PRO 대비 |
|-----|-----|---------|
| 월간 문서 생성 | 200건 | 4배 |
| BLOCK | 전체 12개 | 무제한 |
| 저장 공간 | 50GB | 10배 |
| 팀원 | 10명 | +7명 |
| AI 생성 횟수 | 월 2,000회 | 4배 |

**타겟 고객**

- 중소기업 (직원 10~100명)

## EXPORT 크레딧 시스템

### 설계 원칙

EXPORT 기능은 구독료에 포함하지 않고 **크레딧으로 분리**합니다. 이유는 다음과 같습니다.

1. **모든 고객이 사용하지 않음**: 해외 진출하는 기업만 필요
2. **건당 비용이 높음**: 영문 번역/검수 비용 포함
3. **ARPU 증가 효과**: 필요한 고객에게서 추가 수익 창출

### 크레딧 사용량 정의

```typescript
interface ExportCreditUsage {
  // SAM.gov (미국 정부 조달)
  "sam_gov_basic": 5;           // 기본 제안서 (5크레딧 = 약 5만원)
  "sam_gov_full": 15;           // 풀 패키지 (15크레딧 = 약 10~15만원)
  
  // UNGM (유엔 조달)
  "ungm_basic": 8;              // 기본 입찰서
  "ungm_full": 20;              // 풀 패키지
  
  // 일반 영문 문서
  "translation_per_page": 2;    // AI 번역 (페이지당)
  "expert_review_per_page": 3;  // 전문가 검수 (페이지당)
}
```


### QETTA의 가격 정당화

QETTA PRO가 월 9.9만원인 이유는 다음과 같습니다.

1. **시간 절약 가치**: 8시간 → 30분 = 7.5시간 절약 × 30건/월 = 225시간/월
   - 시간당 인건비 3만원 기준 → 675만원 가치
   - 9.9만원은 가치의 1.5%에 불과

2. **전문성**: 정부지원사업 규정, 양식, 용어를 이미 학습
   - 컨설팅 비용 절감 (건당 50~200만원)

3. **정확도**: 도메인 특화 BLOCK으로 오류율 감소
   - 재작성/수정 비용 절감
