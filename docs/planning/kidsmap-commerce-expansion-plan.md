# KidsMap Commerce Expansion Plan
## 와디즈 모델 기반 공동구매/펀딩 사업 확장 계획

---

## Executive Summary

KidsMap의 MAP-FIRST 플랫폼을 기반으로 와디즈(Wadiz) 스타일의 공동구매/펀딩 모델을 도입하여, 장소 발견(Discovery)에서 구매(Commerce)까지 이어지는 Full-Funnel 플랫폼으로 확장합니다.

### 핵심 가치 제안
- **기존 사용자 기반 활용**: 장소 탐색 사용자 → 입장권/상품 구매 전환
- **데이터 기반 타겟팅**: 연령대(infant/toddler/child/elementary), 관심 카테고리, 방문 이력 활용
- **공동구매 할인**: 그룹 파워를 통한 15-30% 할인 혜택

### 2.4 파트너십 전략

#### 1단계: 중소형 키즈카페 (M1-M3)
- **타겟**: 전국 챔피언 키즈카페 (45개점), 플레이타임, 블랙벨트
- **제안 가치**: 비수기 집객, 신규 고객 유치
- **할인 조건**: 최소 50명 공동구매 시 20% 할인

#### 2단계: 대형 테마파크 (M4-M6)
- **타겟**: 에버랜드, 롯데월드, 키자니아
- **제안 가치**: KidsMap 사용자 데이터 기반 타겟 마케팅
- **할인 조건**: 평일/비수기 한정, 200명+ 공동구매 시 15% 할인

### 2.6 기술 요구사항

#### 데이터베이스 확장
```sql
-- 공동구매 딜 테이블
CREATE TABLE kidsmap_group_deals (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    place_id UUID REFERENCES kidsmap_place_blocks(id),
    title VARCHAR(255) NOT NULL,
    description TEXT,
    original_price INT NOT NULL,
    discount_price INT NOT NULL,
    discount_rate DECIMAL(5,2),
    min_participants INT DEFAULT 10,
    max_participants INT,
    current_participants INT DEFAULT 0,
    start_date TIMESTAMPTZ NOT NULL,
    end_date TIMESTAMPTZ NOT NULL,
    status VARCHAR(20) DEFAULT 'pending', -- pending, active, success, failed
    created_at TIMESTAMPTZ DEFAULT now(),
    updated_at TIMESTAMPTZ DEFAULT now()
);

-- 참여자 테이블
CREATE TABLE kidsmap_deal_participants (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    deal_id UUID REFERENCES kidsmap_group_deals(id),
    user_id UUID NOT NULL,
    quantity INT DEFAULT 1,
    payment_status VARCHAR(20) DEFAULT 'pending',
    payment_key VARCHAR(255),
    order_id VARCHAR(255) UNIQUE,
    created_at TIMESTAMPTZ DEFAULT now()
);

-- 입장권(바우처) 테이블
CREATE TABLE kidsmap_vouchers (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    participant_id UUID REFERENCES kidsmap_deal_participants(id),
    voucher_code VARCHAR(20) UNIQUE NOT NULL,
    qr_code_url TEXT,
    valid_from DATE NOT NULL,
    valid_until DATE NOT NULL,
    used_at TIMESTAMPTZ,
    status VARCHAR(20) DEFAULT 'valid' -- valid, used, expired, refunded
);
```

#### API 엔드포인트
```typescript
// 공동구매 API
POST /api/kidsmap/deals              // 딜 생성 (관리자)
GET  /api/kidsmap/deals              // 딜 목록 조회
GET  /api/kidsmap/deals/:id          // 딜 상세 조회
POST /api/kidsmap/deals/:id/join     // 딜 참여 (결제 연동)
GET  /api/kidsmap/deals/:id/status   // 딜 상태 조회

// 바우처 API
GET  /api/kidsmap/vouchers           // 내 바우처 목록
GET  /api/kidsmap/vouchers/:code     // 바우처 검증 (장소용)
POST /api/kidsmap/vouchers/:code/use // 바우처 사용 처리
```
